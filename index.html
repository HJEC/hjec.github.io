who?

Hey there, I'm Henry J. E. Crookes.

A Full-Stack Web-Developer by trade, U.K native and based in Berlin.

Things I enjoy: reading obscure books, tv-shows involving robots, dogs with expressive faces, and making useful and pointless websites.

what?

Thank you for checking out my site.

This is version 1.0 of my web-developer portfolio.
As such, I wanted to create a site that displayed all the essentials, but with only the basic tech required. This is only a static page, and throwing a bulky framework and extraneous libraries on top seemed excessive. Therefore, I wanted to showcase my solid grasp of the basic web-development skills, and so this entire site is built with pure HTML5, CSS3, Javascript and jQuery.

If you want to see my experience with newer, more impressive technologies such as Vue, React, mySQL, AWS etc, I believe my portfolio projects will speak for themselves.

when?

I have been working with coding for the web for around a year now. In February 2020, I graduated from Spiced-Academy after an intensive 3-month bootcamp to strengthen my skills in fundamental design and programming.

why?

Good question.

Web Development is a challenge, and a creative outlet. I like to create web projects focussed on user interactivity, and unique graphics.

I love to explore creative fields such as music, animation, video and writing. I believe that designing for the web is the perfect focal point of all of my disciplines.

how?

Employing a diverse range of technologies, I create clean and responsive sites, and tools with dynamic effects.

I also work with 3D design, using Blender after 2 years of self study. Using libraries such as Three.js to merge 3D with the web to create distinctive elements is one of my main interests.
